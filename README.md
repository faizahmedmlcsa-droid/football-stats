# ⚽ Football Stats Dashboard

A fancy animated FIFA/Football statistics dashboard — fully built with Python + Chart.js.

## 🚀 How to Run (3 steps)

### Step 1
```
cd fifa_stats
```

### Step 2
```
python analyze.py
```

### Step 3
```
python -m http.server 8000
```

Then open: **http://localhost:8000/dashboard.html**

---

## 📊 What's Inside

- 🥇 Top Scorers & Assisters leaderboard
- 🕸️ Player Radar chart (Pace, Shooting, Passing, Dribbling...)
- 🗺️ Goals by Nation + Position + Competition
- 🏟️ Top Clubs by goal output
- 💥 Highest Scoring Matches
- ⭐ Top Rated Players
- 📋 Sortable full player database (30 players)

## 🛠️ Tech
- Python (zero pip installs)
- Chart.js (CDN)
- HTML + CSS + JS
